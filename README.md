# SvgToPng (macOS / Linux)

Converts a set of .svg- to .png-files.

* choose destination
* choose source
* press 'OK'
* set size
* set color (hex-format)
* start batch


## Requirements

Your system needs 'librsvg' installed.

`$ brew install librsvg`
